## 一、命名规则

| 规则 | 说明 | 示例 |
|------|------|------|
| 格式 | `{NN}-{主题}-{YYYYMMDD}.md`（与总库同名） | `01-rag相关-20260529.md` |
| 连续合并 | 同日且与上一条主题连续 | 续写总库同一文件，同步本目录 |
| 序号 `NN` | **总库** `docs/questions/` 内递增 | `01`、`02` |
| 主题 | 简短中文，概括问题领域 | `rag相关`、`cli-jline补全` |
| 日期 | 答复落盘日 `YYYYMMDD` | `20260529` |

## 三、与问题库的关系

| 类型 | 路径 | 用途 |
|------|------|------|
| **问题总库**（主副本） | `docs/questions/{NN}-{主题}-{YYYYMMDD}.md` | 完整问答 + 关键代码 |
| **模块副本**（本目录） | `questions/{NN}-{主题}-{YYYYMMDD}.md` | 与总库同步；可映射 category 时写入 |
| **问题库索引** | `meta/question-bank-*.md` | 1～3 句摘要 + 链接 |

写入问题记录后，**自动**同步一条摘要到问题库（无需询问）。
