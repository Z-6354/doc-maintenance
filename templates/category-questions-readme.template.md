## 一、命名规则

| 规则 | 说明 | 示例 |
|------|------|------|
| 格式 | `{NN}-{主题}-{YYYYMMDD}.md` | `01-rag相关-20260529.md` |
| 连续合并 | 同日且与上一条主题连续 | 续写同一文件，追加 `## 问答二` |
| 序号 `NN` | 仅新建文件时递增；续写沿用当日锚点文件 NN | `01`、`02` |
| 主题 | 简短中文，概括问题领域 | `rag相关`、`cli-jline补全` |
| 日期 | 答复落盘日 `YYYYMMDD` | `20260529` |
